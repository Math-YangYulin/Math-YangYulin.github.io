定积分的定义

定义 1

设 [a,b][a, b][a,b] 上有 n−1n - 1n−1个点，依次为

a=x0<x1<⋯<xn−1<xn=ba = x_{0} < x_{1} < \cdots < x_{n - 1} < x_{n} = b a=x0​<x1​<⋯<xn−1​<xn​=b

它们把 [a,b][a, b][a,b] 分成 nnn 个小区间 Δi=[xi−1,xi],i=1,2,⋯ ,n\Delta_{i} = [x_{i - 1}, x_{i}], i = 1, 2, \cdots, nΔi​=[xi−1​,xi​],i=1,2,⋯,n. 这些小区间构成对 [a,b][a, b][a,b] 的一个分割，记为

T={Δ1,Δ2,⋯ ,Δn}T = \{ \Delta_{1}, \Delta_{2}, \cdots, \Delta_{n} \} T={Δ1​,Δ2​,⋯,Δn​}

小区间 Δi\Delta_{i}Δi​ 的长度为 Δxi=xi−xi−1\Delta x_{i} = x_{i} - x_{i - 1}Δxi​=xi​−xi−1​，并记

∥T∥=max⁡1≤i≤n{Δxi}\Vert T \Vert = \max_{1\leq i \leq n} \{ \Delta x_{i} \} ∥T∥=1≤i≤nmax​{Δxi​}

称为分割 TTT 的模，或称为分割细度.

定义 2

设函数 fff 在 [a,b][a, b][a,b] 上有定义. 对于 [a,b][a, b][a,b] 的任意一个分割 T={Δ1,Δ2,⋯ ,Δn}T = \{ \Delta_{1}, \Delta_{2}, \cdots, \Delta_{n} \}T={Δ1​,Δ2​,⋯,Δn​}，任取点 ξi∈Δi,i=1,2,⋯ ,n\xi_{i} \in \Delta_{i}, i = 1, 2, \cdots, nξi​∈Δi​,i=1,2,⋯,n，并作和式

∑i=1nf(ξi)Δxi\sum_{i = 1}^{n}f(\xi_{i})\Delta x_{i} i=1∑n​f(ξi​)Δxi​

称此和式为函数 fff 在 [a,b][a, b][a,b] 上的一个积分和，也称黎曼和.

定义 3

设 fff 是定义在 [a,b][a, b][a,b] 上的一个函数，III 是一个确定的实数. 若任给 ε>0\varepsilon > 0ε>0，存在 δ>0\delta > 0δ>0，使得对 [a,b][a, b][a,b] 的任意一个分割 TTT，以及在其上任意选取的点集 {ξi}\{ \xi_{i} \}{ξi​}，只要 ∥T∥<δ\Vert T \Vert < \delta∥T∥<δ，就有

∣∑i=1nf(ξi)Δxi−I∣<ε\left| \sum_{i = 1}^{n}f(\xi_{i})\Delta x_{i} - I \right| < \varepsilon ∣∣∣∣∣​i=1∑n​f(ξi​)Δxi​−I∣∣∣∣∣​<ε

则称函数 fff 在区间 [a,b][a, b][a,b] 上可积或黎曼可积；数 III 称为 fff 在 [a,b][a, b][a,b] 上的定积分或黎曼积分，记作

I=∫abf(x) dxI = \int_{a}^{b} f(x) \ \mathrm{d}x I=∫ab​f(x) dx

其中 fff 称为被积函数，xxx 称为积分变量，[a,b][a, b][a,b] 称为积分区间，a,ba, ba,b 称为这个定积分的下限和上限.

可积条件

根据定积分的定义判断某个函数在某个区间上是否可积，我们需要验证：对于任意分割和其上的任意点集，当分割系度趋于零的时侯，黎曼和都趋于同一个常数. 这是十分复杂的，甚至是不可能完成的. 因此我们需要寻找一个可积的充要条件，使得判断某个函数在某个区间上是否可积更加简便.

可积准则

函数 fff 在 [a,b][a, b][a,b] 上可积的充要条件是：

任给 ε>0\varepsilon > 0ε>0，存在 δ>0\delta > 0δ>0，使得对 [a,b][a, b][a,b] 的任意一个分割 TTT，只要 ∥T∥<δ\Vert T \Vert < \delta∥T∥<δ，就有

S(T)−s(T)=∑i=1nωiΔxi=∑TωiΔxi<εS(T) - s(T) = \sum_{i = 1}^{n} \omega_{i} \Delta x_{i} = \sum_{T} \omega_{i} \Delta x_{i} < \varepsilon S(T)−s(T)=i=1∑n​ωi​Δxi​=T∑​ωi​Δxi​<ε

使用该条件判断可积性摆脱了分割上的任取点集，这已经大大的降低了证明的复杂性，但该使用条件判断可积性时仍然需要对于任意分割成立，因此需要寻找更弱的条件.

上述充要条件的等价表述：

任给 ε>0\varepsilon > 0ε>0，存在对 [a,b][a, b][a,b] 的一个分割 T0T_{0}T0​，使得

S(T0)−s(T0)=∑i=1nωiΔxi=∑T0ωiΔxi<εS(T_{0}) - s(T_{0}) = \sum_{i = 1}^{n} \omega_{i} \Delta x_{i} = \sum_{T_{0}} \omega_{i} \Delta x_{i} < \varepsilon S(T0​)−s(T0​)=i=1∑n​ωi​Δxi​=T0​∑​ωi​Δxi​<ε

该条件判断可积性只需要存在一个分割满足即可，因此可以极大程度上降低证明的复杂性.

可积函数类

若 fff 是 [a,b][a, b][a,b] 上的连续函数，则 fff 在 [a,b][a, b][a,b] 上可积.

若 fff 是 [a,b][a, b][a,b] 上只有有限个间断点的有界函数，则 fff 在 [a,b][a, b][a,b] 上可积.

若 fff 是 [a,b][a, b][a,b] 上的单调函数，则 fff 在 [a,b][a, b][a,b] 上可积.

定理 1

若 T′T'T′ 是 TTT 增加若干个分点后得到的分割，则 ∑T′ωi′Δxi′≤∑TωiΔxi\sum\limits_{T'} \omega_{i}'\Delta x_{i}' \leq \sum\limits_{T} \omega_{i}\Delta x_{i}T′∑​ωi′​Δxi′​≤T∑​ωi​Δxi​.

定理 2

若函数 fff 在 [a,b][a, b][a,b] 上可积，且 [α,β]⊂[a,b][\alpha, \beta] \subset [a, b][α,β]⊂[a,b]，则 fff 在 [α,β][\alpha, \beta][α,β] 上也可积.

定理 3

设 fff 和 ggg 都在 [a,b][a, b][a,b] 上有界，且仅在有限个点 f(x)≠g(x)f(x) \neq g(x)f(x)​=g(x).
若 fff 在 [a,b][a, b][a,b] 上可积，则 ggg 在 [a,b][a, b][a,b] 上也可积，且 ∫abf(x) dx=∫abg(x) dx\displaystyle\int_{a}^{b} f(x) \ \mathrm{d}x = \displaystyle\int_{a}^{b} g(x) \ \mathrm{d}x∫ab​f(x) dx=∫ab​g(x) dx.

定理 4

设 fff 在 [a,b][a, b][a,b] 上有界，{an}⊂[a,b]\{ a_{n} \} \subset [a, b]{an​}⊂[a,b]，lim⁡n→∞an=c\lim\limits_{n \to \infty} a_{n} = cn→∞lim​an​=c.
若 fff 在 [a,b][a, b][a,b] 上只有 an(n=1,2,⋯ )a_{n}(n = 1, 2, \cdots)an​(n=1,2,⋯) 为其间断点，则 fff 在 [a,b][a, b][a,b] 上可积.

定理 5

若 fff 在区间 Δ\DeltaΔ 上有界，则

ω=sup⁡x′,x′′∈Δ∣f(x′)−f(x′′)∣\omega = \sup_{x', x'' \in \Delta} \left|f(x') - f(x'')\right| ω=x′,x′′∈Δsup​∣f(x′)−f(x′′)∣

定理 6

设 fff 在 [a,b][a, b][a,b] 上有定义.
若任给 ε>0\varepsilon > 0ε>0，存在 [a,b][a, b][a,b] 上的可积函数 ggg，使得

∣f(x)−g(x)∣<ε\left| f(x) - g(x) \right| < \varepsilon ∣f(x)−g(x)∣<ε

则 fff 在 [a,b][a, b][a,b] 上可积.

定积分的性质

性质 1（有界性）

若 fff 在 [a,b][a, b][a,b] 上可积，则 fff 在 [a,b][a, b][a,b] 上必定有界.

性质 2（线性性质）

若 fff 和 ggg 都在 [a,b][a, b][a,b] 上可积，则 αf±βg\alpha f \pm \beta gαf±βg 在 [a,b][a, b][a,b] 上也可积，且

∫abαf(x)±βg(x) dx=α∫abf(x) dx±β∫aag(x) dx\int_{a}^{b} \alpha f(x) \pm \beta g(x) \ \mathrm{d}x = \alpha \int_{a}^{b} f(x) \ \mathrm{d}x \pm \beta \int_{a}^{a} g(x) \ \mathrm{d}x ∫ab​αf(x)±βg(x) dx=α∫ab​f(x) dx±β∫aa​g(x) dx

性质 3（乘积可积性）

若 fff 和 ggg 都在 [a,b][a, b][a,b] 上可积，则 fgfgfg 在 [a,b][a, b][a,b] 上也可积.

性质 4（绝对可积性）

若 fff 在 [a,b][a, b][a,b] 上可积，则 ∣f∣|f|∣f∣ 在 [a,b][a, b][a,b] 上也可积，且

∣∫abf(x) dx∣≤∫ab∣f(x)∣ dx\left|\int_{a}^{b} f(x) \ \mathrm{d}x \right| \leq \int_{a}^{b} \left|f(x)\right| \ \mathrm{d}x ∣∣∣∣∣​∫ab​f(x) dx∣∣∣∣∣​≤∫ab​∣f(x)∣ dx

性质 5（区间可加性）

fff 在 [a,b][a, b][a,b] 上可积的充要条件是：
对于任意一点 c∈(a,b)c\in(a, b)c∈(a,b)，fff 在 [a,c][a, c][a,c] 与 [c,b][c, b][c,b] 上都可积，且

∫abf(x) dx=∫acf(x) dx+∫cbg(x) dx\int_{a}^{b} f(x) \ \mathrm{d}x = \int_{a}^{c} f(x) \ \mathrm{d}x + \int_{c}^{b} g(x) \ \mathrm{d}x ∫ab​f(x) dx=∫ac​f(x) dx+∫cb​g(x) dx

性质 6（保序性）

若 fff 和 ggg 都在 [a,b][a, b][a,b] 上可积，且在 [a,b][a, b][a,b] 上恒有 f(x)≥g(x)f(x) \geq g(x)f(x)≥g(x)，则

∫abf(x) dx≥∫abg(x) dx\int_{a}^{b} f(x) \ \mathrm{d}x \geq \int_{a}^{b} g(x) \ \mathrm{d}x ∫ab​f(x) dx≥∫ab​g(x) dx

性质 7（连续点存在性）

若 fff 在 [a,b][a, b][a,b] 上可积，则 fff 在 [a,b][a, b][a,b] 上必存在连续点.

性质 8（严格保序性）

若 fff 和 ggg 都在 [a,b][a, b][a,b] 上可积，且在 [a,b][a, b][a,b] 上恒有 f(x)>g(x)f(x) > g(x)f(x)>g(x)，则

∫abf(x) dx>∫abg(x) dx\int_{a}^{b} f(x) \ \mathrm{d}x > \int_{a}^{b} g(x) \ \mathrm{d}x ∫ab​f(x) dx>∫ab​g(x) dx

积分中值定理

若 fff 在 [a,b][a, b][a,b] 上连续，则存在一点 ξ∈[a,b]\xi \in [a, b]ξ∈[a,b]，使得

∫abf(x) dx=f(ξ)(b−a)\int_{a}^{b} f(x) \ \mathrm{d}x = f(\xi)(b - a) ∫ab​f(x) dx=f(ξ)(b−a)

称 ξ\xiξ 为 fff 在 [a,b][a, b][a,b] 上的中值点.
可以进一步证明中值点 ξ∈(a,b)\xi \in (a, b)ξ∈(a,b).

积分第一中值定理

若 fff 和 ggg 都在 [a,b][a, b][a,b] 上连续，且 g(x)g(x)g(x) 在 [a,b][a, b][a,b] 上不变号，则存在一点 ξ∈[a,b]\xi \in [a, b]ξ∈[a,b]，使得

∫abf(x)g(x) dx=f(ξ)∫abg(x) dx\int_{a}^{b} f(x) g(x) \ \mathrm{d}x = f(\xi) \int_{a}^{b} g(x) \ \mathrm{d}x ∫ab​f(x)g(x) dx=f(ξ)∫ab​g(x) dx

∫abf(x)g(x) dx=g(a)∫aξf(x) dx\int_{a}^{b} f(x) g(x) \ \mathrm{d}x = g(a) \int_{a}^{\xi} f(x) \ \mathrm{d}x ∫ab​f(x)g(x) dx=g(a)∫aξ​f(x) dx

若 ggg 在 [a,b][a, b][a,b] 上单调增加，且 g(x)≥0g(x) \geq 0g(x)≥0，则存在一点 η∈[a,b]\eta \in [a, b]η∈[a,b]，使得

∫abf(x)g(x) dx=g(b)∫ηbf(x) dx\int_{a}^{b} f(x) g(x) \ \mathrm{d}x = g(b) \int_{\eta}^{b} f(x) \ \mathrm{d}x ∫ab​f(x)g(x) dx=g(b)∫ηb​f(x) dx

若 fff 在 [a,b][a, b][a,b] 上可积，ggg 在 [a,b][a, b][a,b] 上单调，则存在一点 ξ∈[a,b]\xi \in [a, b]ξ∈[a,b]，使得

定理 1

微积分基本定理

定理 1

若 fff 在 [a,b][a, b][a,b] 上可积，则 ∫axf(t) dt\displaystyle\int_{a}^{x} f(t)\ \mathrm{d}t∫ax​f(t) dt 在 [a,b][a, b][a,b] 上连续.

定理 2（原函数存在定理）

若 fff 在 [a,b][a, b][a,b] 上连续，则 ∫axf(t) dt\displaystyle\int_{a}^{x} f(t)\ \mathrm{d}t∫ax​f(t) dt 在 [a,b][a, b][a,b] 上可导，且

定理 3（微积分基本定理）

若函数 fff 在 [a,b][a, b][a,b] 上连续，则

∫abf(x) dx=F(b)−F(a)\int_{a}^{b} f(x) \ \mathrm{d}x = F(b) - F(a) ∫ab​f(x) dx=F(b)−F(a)

该定理的结论被称为 Newton-Leibniz 公式.
